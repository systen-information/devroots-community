# 🌿 DevRoots Website

## What's Inside

```
devroots-website/
│
├── index.html        ← The main HTML page (don't edit)
├── package.json      ← Project settings & dependencies
├── vite.config.js    ← Build tool config (don't edit)
├── netlify.toml      ← Tells Netlify how to deploy (don't edit)
├── .gitignore        ← Files Git should ignore (don't edit)
│
└── src/
    ├── main.jsx      ← App startup file (don't edit)
    └── App.jsx       ← ⭐ YOUR WEBSITE CODE (edit this!)
```

## How to Run Locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173
